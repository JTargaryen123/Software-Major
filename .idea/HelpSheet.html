Online Help

Help With Buttons


On the Home Page

Log In Button

The Log In button takes the user to the login page, wherein the user is able to enter a username and password to log in.


Register Button

The Register button takes the user to the register page, wherein the user is able to create a username and password to register for the site.


On the Login Page

Submit Button

The Submit button allows the user to enter their username and password, prompting if incorrect, and taking the user to the next page if correct.


On the Add or Review Page

Add Questions Button

The Add Questions button allows the user to add easy, medium, or hard questions to the existing database of questions.


Review Questions Button

The Review Questions button takes the user to the questions page, allowing the user to start their review of the questions.


On the Questions Page

Easy Button

The Easy button reveals the first easy question and the answer box, allowing the user to enter a question, prompting if correct. The answer needs to be exactly what is written in the database.


Medium Button

The Medium button reveals the first medium question and the answer box, allowing the user to enter a question, prompting if correct. The answer needs to be exactly what is written in the database.


Hard Button

The Hard button reveals the first hard question and the answer box, allowing the user to enter a question, prompting if correct. The answer needs to be exactly what is written in the database.


Easy Answers Button

The Easy Answers button reveals the first easy answer, allowing the user to see what the answer is if they got it wrong.


Medium Answers Button

The Medium Answers button reveals the first medium answer, allowing the user to see what the answer is if they got it wrong.


Hard Answers Button

The Hard Answers button reveals the first hard answer, allowing the user to see what the answer is if they got it wrong.


Next Question Button

The Next Question button reveals the next question to be answered, hiding the answer again and allowing for the user to answer the next question.



Help With Pages

Home Page

The Home Page is where the user can choose to login or register, allowing new users to register and existing users to login.


Login Page

The Login Page is where an existing user is able to login, allowing them to move on with the Subject Helper.


Register Page

The Register Page is where a new user is able to register a username and password into the database, enabling them to login at a later date.


Add Or Review Page

The Add Or Review Page is where a user is able to choose between adding new questions and reviewing questions.


Add Questions Page

The Add Questions Page is where users are able to enter questions into the database, enabling other users to review that question as well.


Review Questions Page

The Review Questions Page is where users are able to review questions through the Subject Helper software, furthering their knowledge on the subject.
